# 🤖 Agentic AI

A collection of Agentic AI projects, workflows, and real-world implementations that demonstrate how autonomous AI agents can reason, plan, use tools, and complete complex tasks with minimal human intervention.

This repository is designed for developers, AI engineers, and students who want to learn and build next-generation AI systems using modern agent architectures, tool calling, workflows, memory, and autonomous decision-making.

---

# 🚀 What is Agentic AI?

Agentic AI refers to AI systems that can:

* 🧠 Reason about tasks
* 🎯 Set goals
* 🛠️ Use external tools
* 🔄 Execute multi-step workflows
* 📚 Maintain memory and context
* 🤝 Collaborate with other agents
* ⚡ Take autonomous actions

Unlike traditional chatbots that simply respond to prompts, Agentic AI systems actively plan and perform actions to achieve objectives. Modern agent frameworks increasingly focus on tool usage, multi-agent collaboration, workflow orchestration, and autonomous execution.

# ⚙️ Installation

## 1. Clone Repository

```bash
git clone https://github.com/Pranshu51/Agentic-AI.git

cd Agentic-AI
```

---

## 2. Create Virtual Environment

### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

### Linux / Mac

```bash
python -m venv venv

source venv/bin/activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Configure Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_api_key

GEMINI_API_KEY=your_api_key

WEATHER_API_KEY=your_api_key
```

---

## 5. Run Agent

```bash
python main.py
```

or run individual examples:

```bash
python 06_Weather_Agent/main.py
```

# 🧠 Agent Workflow

```text
User Goal

     ↓

     ↓
Tool Selection
     ↓
Tool Execution
     ↓
Observation
     ↓
Decision Making
     ↓
Final Response
```

Agent systems typically combine reasoning, planning, and tool execution in iterative workflows.
